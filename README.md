# SpeechRecon
### A script to help your PC recognize the voice using Google on Kali Linux

# Made by ATUL

## Installation Instructions

### Clone the Repo
```bash
git clone https://github.com/Atuls-git/speechRecon.git
```
### Go to the cloned folder
```bash
cd speechRecon
```
### Get permissions
```bash
chmod +x *
```
### Install requirements
```bash
./setup.sh
```
### Run the main file
```bash
python3 main.py
```
## Enjoy :)
